Redux
-----
-> Store -> the global Object containts whole application data / state
-> Reducer -> the function to update / modify the state / store
-> Action -> the instuction to update / fetch the state data to component
-> Dispath -> the instuction
-> ActionTypes -> Type of Action (Interface)
-> InitialState -> default state data
-> StateType -> Type of State of Each Component (Interface)
